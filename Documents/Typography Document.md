* **Primary Display Font**: Poppins.
    * **Explanation**: A geometric, clean font used to ensure visual presence and confidence on dark backgrounds.
    * **Major Use Cases**: Headings (H1/H2), navigation, large UI labels, CTAs, and episode titles.
* **Secondary / Body Font**: Inter.
    * **Explanation**: A highly legible font optimized for screen reading at smaller sizes.
    * **Major Use Cases**: Body paragraphs, metadata, forms, captions, tags, and small UI text.
* **Expressive Accent Font**: Playfair Display (Italic).
    * **Explanation**: An editorial, premium font used strictly for contrast and short emphasis, never for standard reading.
